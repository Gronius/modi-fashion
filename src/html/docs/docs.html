Documentation

Documentation and build capabilities and starter template

Запуск в режимі розработки: gulp
Сбірка оптимізованої версії для продакшена: gulp docs

Можливості стартового шаблона

  • Готові стилі для CSS контейнеров на всі випадки.
  • Міксини та сніппети для адаптива.
  • Міксин та сніппет для CSS retina фонових зображень.

можливості сбірки

  • Автовиправлення шляхів. Сміливо використовуйте підказки редактора під час вказівки шляху до файлу. Складання сама видалить зайві підняття на рівень вгору ../../ у src HTML та background-image у SCSS.
  • Автоматична генерація спрайтів SVG. Докладніше див. нижче.

Особливості збірки для продакшену gulp docs

  • Автоматичне стиск jpg зображень
  • Автоматична генерація webp графіки та вказівки webp формату HTML.
  • При прописуванні тега img не обов'язково прописувати srcset із 2x. Плагін 'gulp-webp-retina-html' у тягу 'html:dev' автоматично додає тег picture та source для 2x зображень. Обов'язково зберігати картинки в роздільній здатності @2x.
